# Final project

## Description
The Nutrition Data Visualization Application is a powerful JavaScript-based tool designed to help you organize, analyze, and visualize nutrition information effortlessly. Whether you're a health-conscious individual, a fitness enthusiast, or a nutritionist, this application empowers you to make data-driven decisions about your diet.

## Setup

**First time**

```bash
git clone https://github.com/jadjoubran/web-starter-parcel.git
cd web-starter-parcel
npm install
```

**Then you can run the server**

```bash
npm run serve
```

Browse to [localhost:1234](http://localhost:1234)

## Parcel

This starter uses parcel (v2) to bundle your scripts. Refer to [jadjoubran/web-starter-parcel](https://github.com/jadjoubran/web-starter-parcel/) and the official [parcel documentation](https://parceljs.org/) for more info.

## Build for production

When you're ready to deploy, you can build for production with the following command:

```bash
npm run build
```
